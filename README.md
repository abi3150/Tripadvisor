

## 📌 Tripadvisor Clone (HTML & CSS)

### 📖 Description

This project is a **frontend clone of the Tripadvisor website**, built using only **HTML and CSS**. It replicates the layout and design of a travel platform where users can explore destinations, hotels, restaurants, and activities.

The goal of this project is to practice **web design fundamentals**, including layout structuring, responsive design, and styling techniques.

---

### 🚀 Features

* 🌍 Homepage with travel search section
* 🏨 Hotel, restaurant, and destination cards
* 🔍 Search bar UI design
* 📸 Image gallery for travel places
* 📱 Responsive layout (mobile-friendly)
* 🎨 Clean and modern UI similar to Tripadvisor

---

### 🛠️ Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout
* **Flexbox / Grid** – Responsive design
